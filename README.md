# Tradeee

Tradeee is a portfolio management system designed for Zerodha and Binance users.

## Current Status
- **Binance Integration**: Currently implemented. The backend handles fetching funding wallet balances and account information using the Binance API.
- **Zerodha Integration**: Planned for future development.
- **Frontend**: Built with Next.js, designed to display the user's trading portfolio.
- **Backend**: Built with Node.js and Express.
